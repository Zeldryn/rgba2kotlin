
# rgba2kotlin
A simple tool to convert RGBA color format into Kotlin Color format. For example, rgba(25,42,12,0.5) will be converted to Color(red=25, green=42, blue=12, alpha=127).

# HOW TO USE? 
1. type or copy your rgba format into text field;
2. the format must be like this:
   - rgba(12,42,51,1)
   - rgba(32,122,52,0.1)
3. press Enter to get the result
4. copy the text and use it in your Kotlin code
