# rgba2kotlin
=======

>>>>>>> 40eab97fab1807a8ee8591e0e1af439060e9152d
A simple tool to convert RGBA color format to Kotlin Color format.
For example, `rgba(25,42,12,0.5)` will be converted to `Color(red=25, green=42, blue=12, alpha=127)`.

# HOW TO USE?
1. Copy or type your RGBA value into the input field.
2. The format must be like this:
   - `rgba(244,21,44,1)`
   - `rgba(42,12,42,0.4)`
3. Press Enter to get the result.
